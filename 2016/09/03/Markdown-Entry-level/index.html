Markdown --入门

发表于 2016-09-03   |   分类于 Markdown   |     |   阅读次数   |  

Markdown让我们专注写作，而不是关注排版。

我接触Markdown的时间并不长，在这里只是列举出部分常用的语法。
我阅读的第一篇与Markdown有关的文章是来自于简书的「献给写作者的 Markdown 新手指南」，第一个感觉就是这么简单的几个标签就OK了？这相对于 HTML 标记语言来说，简直太轻量级了。

Markdown的优点

  • 纯文本内容，可以用各种文本编辑器来打开。
  • 专注于写作而不是排版样式。
  • 转换方便，可以轻松转换为 HTML、PDF等。
  • 标记语法可读性强，学习成本低。 工具

我个人使用的是Mou。 是一款Mac OS X上的免费应用，它支持实时预览，你在左边编辑，在右边可以实时看到效果。同时，在GitHub上还有很多爱好者写的主题也可以使用。

二、Markdown常用语法总结

  1. 标题

  2. 加粗，斜体

  3. 分割线

  4. 链接，图片

  5. 表格

  6. 代码块
    如果你是个程序猿，需要在文章里优雅的引用代码框，在 Markdown下实现也非常简单，只需要用两个 ` 把中间的代码包裹起来。使用TAB进行缩进。

  7. 引用

  8. 脚注

  9. 创建链接

  10. 转义
